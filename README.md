# A.R.I.A. - Adaptive Responsive Intelligence Assistant

A.R.I.A. is an advanced, autonomous AI assistant capable of deep-level PC control, dynamic coding, browser automation, and contextual memory. Inspired by JARVIS, A.R.I.A. features a high-fidelity TypeScript frontend and a powerful Python/FastAPI backend.

## Features

- **Contextual Memory**: ChromaDB-powered RAG memory retains cross-session conversation history, system states, and tool outputs.
- **Deep System Control**: Execute shell commands, manage files, and interact with the host OS natively.
- **Advanced Coding Sandbox**: Execute Python and JavaScript/Node.js code dynamically, perform surgical file edits, and search across codebases.
- **Browser Automation**: Autonomously navigate the web, research topics, and interact with complex web UI.
- **High-Fidelity Interface**: Real-time WebSocket-based frontend built with TypeScript and Vite.
